## Open Market Project
Commerce 가 제공하는 기능들을 직접 구현함으로써 대규모 트래픽, 대용량 데이터 처리 등을 통해 Commerce 도메인을 이해하자

## 기능 
- 회원관리 
- 장바구니 (스케쥴링 처리)
- 메인화면 캐싱 
- 상품조회, 구매
- 관리자페이지 (상품등록, 광고비에 따른 상품노출)
- 이벤트(남/여, 연령 등 카테고리화 -> 선착순 이벤트 )
